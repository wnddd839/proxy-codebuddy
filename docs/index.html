CodeBuddy Proxy

专注 CodeBuddy 的自建反代。OAuth 登录后协议直连，对外给出标准 OpenAI 兼容 `/v1`，可直接接 NewAPI。

打开仓库 快速开始

Only CodeBuddy

一件事做到位

不是多厂商聚合，也不是 Cursor 网关。这个项目只服务 CodeBuddy 账号 → OpenAI 兼容接口。

01

协议直连

默认 protocol_direct。OAuth 后直连上游，模型列表走 /v3/config，不必再挂 CLI serve。

02

真实 Credits

管理台读官网用量接口，展示剩余 / 总额。一眼知道账号还剩多少。

03

账号池 + 管理台

多账号轮询、启用禁用、刷新 token。写盘有保护，刷新失败不会把账号列表清空。

04

OpenAI 形状

GET /v1/models、POST /v1/chat/completions。NewAPI、自建客户端，按熟悉方式接入。

Get started

三步跑起来

Node 20+。克隆、写密钥、npm start。默认国内站 codebuddy.cn。

范围说明：本项目只做 CodeBuddy 反代，不做 Cursor、也不做多厂商聚合。 
# clone & configure
git clone https://github.com/wnddd839/proxy-codebuddy.git
cd proxy-codebuddy && cp .env.example .env

# start
npm start

# endpoints
API    http://127.0.0.1:32126/v1
Admin  http://127.0.0.1:32126/direct-admin/

Ship it

把 CodeBuddy 接到你的工具链

管理台完成 OAuth，客户端只认 Base URL 和 API Key。完整说明见仓库 README。
